Employee Attrition Prediction & Risk Scoring System

Project Overview:

This project is an HR Analytics Dashboard built using Python and Streamlit. 
It helps analyze employee attrition and provides visual insights through interactive charts and metrics.

Features:

* Employee Dashboard
* Attrition Analysis
* Risk Analysis
* Department Analysis
* Model Performance

Technologies Used:

* Python
* Streamlit
* Pandas
* Plotly
* Scikit-learn
